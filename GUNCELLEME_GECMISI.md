# 🚀 YT Downloader - Güncelleme Geçmişi (Changelog)

Projede SemVer (Semantic Versioning: *MAJÖR.MİNÖR.YAMA*) standardı uygulanmaktadır.

---

### 📌 v2.0.1 (Mevcut Sürüm) - *03.07.2026*
- 🎯 **Otomatik Ekran Ortalaması**: Pencere açıldığında ekranın altında kalması engellendi; açılır açılmaz ekranın tam ortasına (`window.center()`) konumlandırılması sağlandı.
- 📋 **Pano Yapıştırma (Yapıştır Butonu) Onarıldı**: Windows Pano kilitlerine karşı 10 denemeli (retry-loop) koruma ve `navigator.clipboard` yedek mekanizması eklendi. Panodan yapıştırma butonuna tıklandığında link anında yapışıyor.
- 🏷️ **Standart SemVer Sürüm Numaralandırması**: Küçük yama ve hata düzeltmeleri için `v2.0.1`, `v2.0.2` gibi standart yama (patch) sürümleri kullanılacaktır.

### 📌 v2.0.0 - *02.07.2026*
- 🖥️ **PyWebView & Tailwind CSS Mimarisine Geçiş**: Modern PyWebView masaüstü penceresi, Tailwind CSS karanlık teması, 1080p MP4 birleştirme, MP3 ID3 kapak resmi gömme ve ultra hızlı RAM önbellekli indirme geçmişi.
